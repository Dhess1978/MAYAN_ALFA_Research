# MAYAN_ALFA Research

Public research and archival layer for the MAYAN_ALFA benchmark framework.

This public release contains:

- public-safe publication package,
- QA verification package,
- release documentation,
- benchmark summaries,
- public visualization and graph layers,
- governance and archival materials.

---

## Public Release Boundary

This repository contains:

- public-safe benchmark evidence only,
- reproducible public research artifacts,
- governance and QA documentation,
- DOI archival synchronization materials.

This repository does NOT contain:

- protected validation archives,
- protected internal workflow evidence,
- private source-core systems,
- internal heuristics,
- optimization internals,
- private observation registries,
- local machine paths or host identifiers.

---

## Purpose

This repository is intended for:

- GitHub public publication,
- public review and transparency,
- reproducible research documentation,
- citation and DOI archival synchronization,
- long-term public archival preservation.

## Governance Notice

MAYAN_ALFA Research operates under a strict public/protected governance boundary.

Public releases are intentionally separated from protected-core systems, internal validation infrastructure, private heuristics, and restricted research environments.

All public artifacts are curated to preserve reproducibility, citation integrity, and long-term archival transparency without exposing protected internal systems.
